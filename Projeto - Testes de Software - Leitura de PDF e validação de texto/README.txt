Projeto de leitura de pdf e validação de conteúdo!

Usei a biblioteca PyPDF2 do python para realizar a leitura do pdf e a extração do texto para um arquivo txt. Após isso, é possível realizar uma validação de conteúdo através do próprio Robot Framework.

Com essa validação é possível automatizar cenários onde PDFs são baixados e confirmar os seus dados :)
